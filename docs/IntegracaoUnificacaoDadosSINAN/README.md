# Integração e Unificação de Dados - SINAN

Esta pasta consolida a task `Integração e Unificação de Dados - SINAN`.

## Objetivo

Integrar a base semanal processada do SINAN com a base climática real do INMET, produzindo um dataset unificado, consistente e pronto para a próxima etapa de modelagem.

- `evidencias/`

## Observação

Nesta task, a integração foi feita no nível de `semana epidemiológica`, preservando:

- o recorte real do SINAN já validado
- o calendário semanal do INMET
- os clusters e atributos derivados do SINAN
- a coerência temporal com a base unificada entregue no `TCC1`
